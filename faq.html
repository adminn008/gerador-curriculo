PERGUNTAS FREQUENTES

O CVFLASH é realmente gratuito?

Sim. O projeto foi criado para ser uma ferramenta social. Não cobramos taxas para gerar ou baixar o currículo em PDF. As doações via PIX são totalmente opcionais e servem para manter o site no ar e financiar a futura v2.0.

Meus dados ficam salvos no sistema?

Segurança máxima: NÃO salvamos nada. O CVFLASH processa os dados localmente no seu navegador. Quando você fecha a aba, os dados desaparecem para sempre. Você tem controle total.

Como baixar o currículo pelo celular?

Ao clicar no botão de gerar, seu navegador abrirá a tela de impressão. No Android ou iPhone, basta selecionar a opção "Salvar como PDF". O arquivo irá direto para a sua pasta de downloads.
